# TODO API

- /todo (POST) create todo
- /todo (GET) get All todo
- /todo/<id> (GET) get todo by ID
- /todo/<id> (PUT/PATCH) update todo
- /todo/<id>/finish finish todo
- /todo/<id> (DELETE) soft delete todo
